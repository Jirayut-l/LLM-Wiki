## ⚙️ 1. LLM Output Configuration (การตั้งค่าการตอบกลับ)
ก่อนจะเริ่ม Prompt เราต้องเข้าใจ "ปุ่มหมุน" (Parameters) หลักๆ 4 ปุ่มนี้ก่อนครับ:

### 🌡️ Temperature (ความร้อนแรง/ความคิดสร้างสรรค์)
- **0 (Greedy):** ตอบแบบตรงไปตรงมาที่สุด เหมาะสำหรับงาน Logic, คณิตศาสตร์, หรือการสกัดข้อมูล (Data Extraction)
- **0.7 - 1.0:** เริ่มมีความคิดสร้างสรรค์มากขึ้น เหมาะสำหรับการเขียนบทความหรือระดมสมอง (Brainstorming)
- **💡 Pro Tip:** ถ้าใช้ในแอปพลิเคชันสาย Developer แนะนำให้เริ่มที่ **0** เพื่อความแม่นยำครับ!

### 🎯 Top-K & Top-P (การเลือกคำ)
- **Top-K:** เลือกจากกลุ่มคำที่น่าจะเป็นไปได้มากที่สุด K อันดับแรก (เช่น K=40)
- **Top-P (Nucleus Sampling):** เลือกจากกลุ่มคำที่มีความน่าจะเป็นสะสมรวมกันไม่เกิน P (เช่น P=0.95)
- **David's Analogy:** เหมือนเราคัดเลือกนักแสดงครับ Top-K คือคัด 40 คนแรกมาแคสต์ ส่วน Top-P คือคัดจนกว่าจะได้กลุ่มที่คะแนนรวมกันแล้วอุ่นใจที่สุดครับ

### 📏 Max Output Tokens
- ควบคุมความยาวของการตอบกลับ เพื่อประหยัด Token และค่าใช้จ่ายครับ

## 💻 2. Code Prompting (คู่มือสำหรับนักเขียนโค้ด)
Gemini เก่งเรื่องโค้ดมากครับ! แต่เราต้องสั่งให้ถูกจุด:

### ✍️ การเขียนโค้ด (Writing Code)
- **DO:** ระบุภาษา, Framework, และรายละเอียดงานให้ชัดเจน
- **Example:** "จงเขียนสคริปต์ Bash สำหรับ Rename ไฟล์ทั้งหมดใน Folder โดยให้เติม 'draft_' ข้างหน้าชื่อไฟล์"

### 🔍 การอธิบายโค้ด (Explaining Code)
- ส่งโค้ดที่คุณไม่เข้าใจให้ Gemini แล้วใช้บทบาท **"Senior Developer"** ให้เขาอธิบายทีละ Step ครับ

### 🔄 การแปลงภาษา (Translating Code)
- คุณสามารถสั่งให้ "Translate Bash code นี้เป็น Python" ได้เลย! สะดวกมากสำหรับการ Refactor งานเก่าๆ ครับ

### 🐞 การดีบัก (Debugging & Review)
- เมื่อโค้ด Error ให้ส่ง Error Message พร้อมโค้ดเดิมให้ Gemini
- **Pro Way:** "Debug what's wrong and explain how I can improve the code." Gemini จะช่วยหาจุดที่ Logic ผิด (เช่น ลืมแก้ชื่อฟังก์ชัน) ให้ด้วยครับ

## 🚀 Best Practices สำหรับ Dev
1. **Use Instructions over Constraints:** บอก AI ว่า "จงทำอะไร" (Do) ได้ผลดีกว่าบอกว่า "ห้ามทำอะไร" (Don't)
2. **Use Variables:** ในแอปพลิเคชัน ให้ใช้ Placeholder เช่น `{{city}}` เพื่อให้ Prompt นำกลับมาใช้ใหม่ได้ง่าย (Reusable)
3. **Markdown Format:** สั่งให้ AI ส่งผลลัพธ์เป็น JSON หรือ Markdown เพื่อให้เรานำไป Parse ต่อในโปรแกรมได้ง่ายๆ ครับ
