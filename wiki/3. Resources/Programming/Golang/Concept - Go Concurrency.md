# Concept - Go Concurrency

Concurrency is a core feature of Go, primarily implemented through Goroutines and Channels.

## Key Primitives
- **Goroutines**: Lightweight threads managed by the Go runtime.
- **Channels**: Typed conduits through which you can send and receive values.
- **Select**: Statement used to wait on multiple channel operations.
- **Sync Package**: Mutexes, WaitGroups, and other synchronization primitives.
