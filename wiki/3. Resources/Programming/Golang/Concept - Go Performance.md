# Concept - Go Performance

Go provides extensive tools for measuring and improving the efficiency of code.

## Areas of Focus
- Benchmarking (`testing` package)
- Profiling (`pprof`)
- Trace tool
- Memory allocation and GC tuning
- Fuzz testing (introduced in 1.18/1.19)
